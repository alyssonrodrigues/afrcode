Este sistema é compatível com os navegadores: Internet Explorer versão 9 ou superior, Google Chrome e Mozilla Firefox.

Certifique-se de que o "Modo do Navegador" do Internet Explorer seja "Internet Explorer 9" ou superior. Para isto, acione a tecla "F12" e verifique o conteúdo do campo "Modo do Navegador".

Se necessário, atualize o seu navegador para o correto funcionamento deste sistema.
